# Excel Lassonde Website

##  Overview

The front end of Excel Lassonde's website. The project uses React as the JavaScript framework and TailwindCSS as the CSS framework of choice.

The project was bootstrapped using Create React App.

## Requirements

- [NodeJS](https://nodejs.org/en/download/)

## Getting Started

1) Clone repository 
2) Run `npm install` to install dependancies

Note: Run `npm start` to open live server and develop. Take a look at `README.cra.md` for more information.

## Additional Tooling

- [Visual Studio Code](https://code.visualstudio.com/download)
- [Tailwind CSS IntelliSense](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss)
